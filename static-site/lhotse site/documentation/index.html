Booard - Corporate HTML5 Template

  • Created: 8/May/2015
  • Latest update: 8/May/2015
  • By: Theme-Paradise
    • support@theme-paradise.com

Booard HTML5 template can be used to build Corporate, Agency, Personal Portfolio, etc. This HTML5 template implemented beautiful design of original PSD template, and utilizes HTML5 semantics for SEO optimization. Most of animations in this templates are based on CSS3 to improve the performance of the template.

Features

  • 8 Homepage variants
  • Fully Responsive Layout
  • HTML5 Semantics
  • 3 Blog archive page styles
  • 3 Portfolio archive page styles
  • Blog and Portfolio Single Pages
  • Based on Twitter Bootstrap (1170px)
  • Clean design
  • Google Fonts used
  • Stylesheets Built Using LESS CSS compiler
  • And more…

Contents

You should find the following files and folders in the download pack:
  • bootstrap/
  • css/
  • fonts/
  • images/
  • js/
  • less/

bootstrap/ - contains bootstrap v3.3.4 version. It is unmodified, and you can replace it with any version as long as compatibility with our template is maintained.
css/ - LESS-compiled CSS files are here.
fonts/ - fonts used in our templates are here. Currently only FontAwesome icon font is in this folder.
images/ - images in our templates are here.
js/ - Javascript files are residing in this folder.
less/ - We created stylesheet codes in LESS format for better maintenance and future changes, and you can find all LESS files here.
about.html - About page.
